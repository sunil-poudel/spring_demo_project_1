1. go to start.spring.io
2. create new spring project, select maven, java, give name, add
web dependency, select language version and then download zip by clicking generate.
3. open the project in intellij (or any other ide).

4. I recommend creating new spring boot project inside intellij itself instead of
from start.spring.io so everything is ok.
to do so, create new spring boot project in intellij,
    then add dependencies web and dev-tools in creation dialog.
    dev-tools will allow auto reloading the page on save.
