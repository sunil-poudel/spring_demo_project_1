1. go to start.spring.io
2. create new spring project, select maven, java, give name, add
web dependency, select language version and then download zip by clicking generate.
3. open the project in intellij (or any other ide).

4. add dependency in pom.xml (first allow project to build automatically in compiler settings)
        <dependency>
			<groupId>org.springframework.boot</groupId>
			<artifactId>spring-boot-devtools</artifactId>
		</dependency>
